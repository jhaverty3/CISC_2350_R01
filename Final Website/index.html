What is Data Mining?

Data Mining is a shorthand phrase for a part of what is known as the Knowledge Discovery Process.
It involes using mathematical algorithms to find useful patterns in extremely large datasets that are typically
collected by Internet of Things (IOT) devices. The patterns that are found are then interpreted by people to gain useful information.

Why is it Useful?

Data Mining is useful because it is able to harness computing power and statistical
methods simeltaneously to quickly find patterns that would take human investigators long periods of time to do by
hand. This is particularly useful in the field of medicine because of
personal genomics revolution and the large datasets that have accompanied it.

How is it Accomplished?

Data mining is accomplished using several different algorithmic methods discussed elsewhere on this
site. Major examples include Classification, Regression, Clustering and Association Rule Mining.
